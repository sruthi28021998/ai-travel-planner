This is a web application that helps users plan their trips using the power of AI.

Features
Customized Itineraries: Generate travel plans based on your interests and budget.

Intelligent Recommendations: Uses AI to provide personalized travel suggestions.

Setup Instructions
To run this project locally, follow these steps:

Clone the repository:

Bash
git clone https://github.com/sruthi28021998/ai-travel-planner.git
2. **Install dependencies:**
   Navigate into the `frontend` and `backend` directories and run:
   ```bash
npm install
Environment Variables:
Create a .env file in the root directory (or respective folders) and add your API keys:

OPENAI_API_KEY=your_openai_key_here

MONGODB_URI=your_mongodb_connection_string

How to use
After starting both the frontend and backend servers, you can input your destination, travel dates, and budget preference to receive a custom travel plan.
